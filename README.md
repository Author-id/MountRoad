## MountRoad
#### Basic 2D Platformer with animated objects

### Clone this repository:
```console
  git clone https://github.com/Author-id/MountRoad.git
```

### Install plugins:
```console
  pip install -r requirements.txt
```

### Try game in your IDE:
```console
  python main.py
```

### Try game on your computer:
```bash
  use main.exe file
```
